Roles

Roles & responsibilities.

@owner

Owner

Sets direction, approves major changes, protects long-term community health.

@community_admin

Community Admin

Structure, announcements, and day-to-day ops. Keeps the community readable.

@server_admin

Server Admin

World health, maintenance, setup, and stability.

@moderator

Moderator

Keeps the space welcoming, fair, and consistent. Handles conduct and de-escalation.

@event_lead

Event Lead

Runs convoy nights, launch events, showcases, and community sessions.

@support

Support

Helps new people get settled — roles, setup, and first questions.

Routing

Where to take an issue.

Server problem

Support or server admins — setup, lag, outages, version mismatches.

Behavior issue

Moderators — harassment, rule problems, fairness checks.

Event question

Event leads — convoy timing, launch nights, showcase scheduling.

Big-picture idea

Admins & leadership — community ideas, partnerships, feedback.
