This folder should contain images for all game servers. Images should be in JPG format with dimensions around 800x600 pixels for best quality.

1. **minecraft.jpg** - Minecraft server image
2. **starrupture.jpg** - StarRupture server image

8. **nightingale.jpg** - Nightingale server image

10. **squad.jpg** - Squad server image
11. **terraria.jpg** - Terraria server image
12. **fivem.jpg** - FiveM (GTA V) server image

## Image Guidelines:

- **Format**: JPG or PNG
- **Recommended size**: 800x600 pixels (or similar 4:3 ratio)
- **File size**: Keep under 500KB for faster loading
- **Content**: Use official game artwork, screenshots, or promotional images
- **Style**: High quality, eye-catching images that represent each game well

## Where Images Are Used:

- **Home Page** (index.html): Shows 4 featured servers (Minecraft, BeamMP, Palworld, Sons of the Forest)
- **Servers Page** (servers.html): Shows all 12 game servers with detailed information

## Fallback:

If images are not added, the placeholder will show a gradient background with an orange border. The website will still function, but adding actual game images will significantly improve the visual appeal.
